# Beckett Pizza Plaza – 4-for-3 Offer

## Description
This Python program calculates the total price of a pizza order using Beckett Pizza Plaza’s 4-for-3 offer.
The customer orders four pizzas, and the cheapest pizza is free.

The program reads pizza prices from a text file, applies the discount, and displays the final total and
discount percentage.

## How to Run
1. Open a terminal
2. Navigate to the project folder
3. Run the program using:

   python pizza_offer.py sample_order.txt

## Input File Format
- One pizza price per line
- Exactly four prices are required
- Prices must be positive numbers

## Example Output
Order Total is £47.97, a fabulous discount of 19%!
